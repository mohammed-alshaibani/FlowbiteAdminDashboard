# لوحة تحكم Flowbite - قالب مجاني مفتوح المصدر

## نظرة عامة
لوحة تحكم احترافية مبنية باستخدام Tailwind CSS وإطار عمل Flowbite. تم تصميم هذه اللوحة لتكون نقطة بداية رائعة لمشاريع الويب الحديثة مع واجهة مستخدم أنيقة وسهلة التخصيص.

## المميزات

- **تصميم عصري وأنيق** مع واجهة مستخدم سهلة الاستخدام
- **متجاوب بالكامل** مع جميع أحجام الشاشات
- **مبنية باستخدام Tailwind CSS** مع إضافات Flowbite
- **مكونات جاهزة** للاستخدام الفوري
- **ألوان متعددة** مع دعم الوضع المظلم
- **سهلة التخصيص** لتلائم احتياجاتك
- **خفيفة وسريعة** الأداء

## المكونات المتوفرة

- **لوحة القيادة** - نظرة عامة على الإحصائيات والمقاييس
- **المنتجات** - إدارة كتالوج المنتجات
- **العملاء** - إدارة قاعدة عملائك
- **الطلبات** - تتبع وإدارة الطلبات
- **المخزون** - إدارة المخزون والمستودعات
- **التقارير** - تحليلات وإحصائيات متقدمة
- **الإعدادات** - تخصيص إعدادات التطبيق

## المتطلبات التقنية

- Node.js 14.x أو أحدث
- npm أو Yarn
- Hugo (للتنمية المحلية)

## التثبيت

1. استنسخ المستودع:
   ```bash
   git clone https://github.com/themesberg/flowbite-admin-dashboard.git
   cd flowbite-admin-dashboard
   ```

2. قم بتثبيت التبعيات:
   ```bash
   npm install
   # أو
   yarn install
   ```

3. قم بتشغيل بيئة التطوير:
   ```bash
   npm start
   # أو
   yarn start
   ```

4. افتح المتصفح على العنوان:
   ```
   http://localhost:1313/
   ```

## البناء للإنتاج

لإنشاء نسخة إنتاجية من المشروع:

```bash
npm run build
# أو
yarn build
```

سيتم إنشاء الملفات في مجلد `public`.

## التخصيص

### الألوان
يمكنك تخصيص الألوان من خلال تعديل ملف `tailwind.config.js`:

```javascript
theme: {
  extend: {
    colors: {
      primary: {
        50: '#f0f9ff',
        100: '#e0f2fe',
        // ...
      }
    }
  }
}
```

### المكونات
جميع المكونات متوفرة في مجلد `src/components/` ويمكن تعديلها بسهولة.
